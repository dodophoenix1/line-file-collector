🛠️ วิธีการตั้งค่า LINE Bot & Google Drive Integration

ตั้งค่าการทำงานร่วมกันระหว่าง Messaging API และระบบอัปโหลดขึ้น Google Drive คลาวด์อัตโนมัติ

1

ตั้งค่า LINE Bot (Webhook)

ลงทะเบียน LINE Developers Console, ตั้งค่า Webhook URL ชี้ไปที่เซิร์ฟเวอร์ของคุณ (สร้าง HTTPS Tunnel ผ่าน ngrok http 3000) และเปิดสวิตช์ "Use Webhook" ใน LINE Developer Console

2

เปิดใช้งาน Google Drive API & สร้าง Service Account

เนื่องจากนี่คือบอทหลังบ้าน (Backend Service) เราจะใช้ **Service Account** ในการทำงานร่วมกับ Google Drive โดยไม่ต้องกดล็อกอินหน้าเว็บ:

  • ไปที่ Google Cloud Console แล้วเปิดใช้บริการ **Google Drive API** ในโปรเจกต์ของคุณ
  • เข้าเมนู **IAM & Admin -> Service Accounts** สร้างบัญชี Service Account ใหม่ขึ้นมา
  • กดสร้างคีย์ของ Service Account นั้น เลือกประเภทไฟล์เป็น **JSON** ระบบจะดาวน์โหลดไฟล์คีย์ลงเครื่อง
  • ย้ายไฟล์คีย์นี้มาไว้ในโฟลเดอร์โปรเจกต์ของคุณ และตั้งชื่อไฟล์ว่า google-credentials.json
3

สร้างโฟลเดอร์และแบ่งปันสิทธิ์การเข้าถึงใน Google Drive

บอทต้องการสิทธิ์เขียนไฟล์ลงโฟลเดอร์ที่คุณต้องการบน Google Drive:

  • สร้างโฟลเดอร์ใน Google Drive ของคุณเอง
  • แชร์ (Share) โฟลเดอร์นั้นให้กับอีเมลของ Service Account (มีนามสกุลลงท้ายด้วย @...gserviceaccount.com พบในไฟล์ JSON) โดยกำหนดสิทธิ์เป็น **Editor (ผู้แก้ไข)**
  • คัดลอก **Folder ID** จากแถบ URL ของบราวเซอร์ (เช่น URL คือ drive.google.com/drive/folders/1A2B3C... ค่า ID คือรหัสหลังสุด 1A2B3C...)
4

กรอกข้อมูลตัวแปรลงในไฟล์ .env

เปิดไฟล์ .env ของโปรเจกต์แล้วกรอกข้อมูล LINE และ Google Drive Folder ID ดังนี้:

LINE_CHANNEL_ACCESS_TOKEN=วาง Channel Access Token จริง
LINE_CHANNEL_SECRET=วาง Channel Secret จริง

GOOGLE_DRIVE_FOLDER_ID=วาง Google Drive Folder ID ที่คัดลอกมา

อย่าลืม: รีสตาร์ทเซิร์ฟเวอร์โปรเจกต์ของคุณด้วยคำสั่ง npm run dev เพื่อเริ่มใช้การเชื่อมต่อ Google Drive

เรียบร้อย! เมื่อคุณส่งไฟล์เข้าไปในไลน์บอท บอทจะบันทึกไฟล์นั้นเก็บในคอมพิวเตอร์นี้ พร้อมทั้งยิงขึ้น Google Drive ของคุณให้ทันทีพร้อมกับสร้างลิงก์พรีวิวหน้าเว็บนี้โดยอัตโนมัติ!
